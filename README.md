## 🗂️ Mini Projects

### 1. 🖥️ My First Webpage — `C1`
My very first HTML page. Includes a heading, paragraph, image, and a hyperlink to a favourite website.

**Concepts:** Basic HTML tags (`h1`, `p`, `img`, `a`), page structure

---

### 2. 📋 Contact Form — `C2`
A simple contact form with fields for name, email, phone number, subject, and message, plus a newsletter checkbox and submit button.

**Concepts:** HTML forms, `input` types (`text`, `email`, `tel`), `textarea`, `checkbox`, `button`

---

### 3. 🎬 Movie Collection Table — `C2`
An HTML table displaying a list of favourite movies with columns for Title, Genre, Year, and Director.

**Concepts:** HTML tables, `thead`, `tbody`, `th`, `td`, `caption`

---

### 4. 📝 Weekend Plans List — `C2`
A structured page with nested lists showing weekend plans across categories like Outdoor Activities, Movies to Watch, and Foods to Try.

**Concepts:** Unordered lists (`ul`), ordered lists (`ol`), nested lists

---

### 5. 🧱 Semantic About Me Page — `C3`
A simple personal page built using semantic HTML elements with a nav bar, article, section, and footer.

**Concepts:** Semantic HTML (`header`, `nav`, `section`, `article`, `footer`, `div`)

---

### 6. 🥞 Recipe Page — `C4`
A styled recipe page for Pancakes with an image, ingredients list, and numbered instructions. CSS is in a separate file.

**Concepts:** Linking external CSS, ID selectors, styling lists, `header` and `footer`, colour and padding

---

### 7. 🔷 CSS Shapes — `C6`
A set of individual pages, each drawing a different geometric shape using only CSS — no images, no SVG.

| File | Shape | Technique Used |
|---|---|---|
| `circle.html` | 🔴 Red Circle | `border-radius: 50%` |
| `rectangle.html` | Rectangle | `width` and `height` |
| `square.html` | Square | Equal `width` and `height` |
| `triangle.html` | 🟣 Purple Triangle | CSS border trick |
| `diamond.html` | 🔷 Teal Diamond | `transform: rotate(45deg)` |

**Concepts:** CSS box model, `border-radius`, `transform`, the CSS border triangle trick

---

### 8. 👤 Responsive Profile Card — `C7`
A profile card displaying a photo, name, short bio, and action buttons (Contact & Portfolio). Adapts layout on small screens using a media query.

**Concepts:** CSS Flexbox, `border-radius`, `box-shadow`, `object-fit`, `@media` queries, hover transitions

---

### 9. 🐶 Responsive Dog Photo Gallery — `C8`
A photo gallery of 9 dog images arranged in a responsive CSS Grid that shifts from 1 column on mobile → 2 columns on tablet → 3 columns on desktop.

**Concepts:** CSS Grid, `grid-template-columns`, `repeat()`, responsive breakpoints with `@media`, `object-fit: cover`, hover effects with `transform` and `box-shadow`

---

### 10. 🗃️ CSS Grid + Flexbox Layout — `C8`
A structural layout exercise combining CSS Grid and Flexbox to build a page with a header row and a three-column main content area (left panel, split centre column, right panel).

**Concepts:** CSS Grid (`grid-template-rows`), Flexbox (`flex-grow`, `flex-basis`, `flex-direction`), `box-sizing`, nested layouts

---

### 11. 🎡 Animated Ferris Wheel — `C10`
A fully CSS-animated spinning Ferris wheel with 6 spokes and 6 red cabins that rotate continuously around the wheel.

**Concepts:** CSS `@keyframes`, `animation`, `transform-origin`, `nth-of-type()`, `position: absolute`, `rotate()`

---

### 12. 🌕 Animated Earth & Moon — `C10`
A space animation on a black background where a blue Earth spins in the centre and a white Moon orbits around it — all done purely with CSS.

**Concepts:** CSS `@keyframes`, chained `transform` (rotate + translateX + counter-rotate), `position: absolute`, `border-radius`, `overflow: hidden`

---
